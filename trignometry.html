⌨️ Keyboard Shortcuts

  • 0 – 9 → Enter digits
  • . → Decimal point

Operators

  • + → Addition
  • - → Subtraction
  • * → Multiplication
  • / → Division
  • ^ → Power (xʸ)

Scientific Functions

  • s → sin(
  • c → cos(
  • t → tan(
  • l → ln(
  • Shift + L → log(
  • p → π (pi)

Parentheses

  • ( → Left parenthesis
  • ) → Right parenthesis

Actions

  • Enter → Calculate (=)
  • Backspace → Delete last character
  • Escape → Clear (AC)
